# sakura-comic-vue

动漫网站前端部分

1. 一个用于追番的动漫网站，实现了动漫视频的更新与展示，用户注册和登录，评论和回复，视频收藏的功能
2. 技术栈，前端： vue3 + vite + element plus, 后端：flask + gunicorn + mysql 
3. 预览： http://139.196.138.236:8000/#/
